# Tetris Game

## Overview
A classic Tetris game built with vanilla HTML, CSS, and JavaScript. No build system or framework required.

## Project Structure
- `index.html` - Main HTML page with responsive 2-column layout
- `style.css` - Styling with gradient background, modern design, and responsive breakpoints
- `tetris.js` - Game logic including piece movement, rotation, collision detection, and scoring

## Responsive Design
- Large screens (600px+): 2-column layout with game on left, controls on right
- Small screens: Single column layout with controls below game

## How to Run
The project is served as static files using `serve` on port 5000.

## Game Controls
- Arrow Left/Right: Move piece
- Arrow Down: Soft drop
- Arrow Up: Rotate
- Space: Hard drop
- P: Pause/Resume

## Deployment
Configured for static deployment serving the root directory.
