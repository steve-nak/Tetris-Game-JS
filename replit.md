# Tetris Game

## Overview
A classic Tetris game built with vanilla HTML, CSS, and JavaScript. No build system or framework required.

## Project Structure
- `index.html` - Main HTML page
- `style.css` - Styling with gradient background and modern design
- `tetris.js` - Game logic including piece movement, rotation, collision detection, and scoring

## How to Run
The project is served as static files using `serve` on port 5000.

## Game Controls
- Arrow Left/Right: Move piece
- Arrow Down: Soft drop
- Arrow Up: Rotate
- Space: Hard drop
- P: Pause/Resume

## Deployment
Configured for static deployment serving the root directory.
