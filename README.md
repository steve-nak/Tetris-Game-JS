# Tetris Game

A simple, classic Tetris game built with HTML, CSS, and JavaScript.

## How to Play

1. Open `index.html` in your web browser
2. Click "Start Game" to begin. Use the button to Pause/Resume anytime.
3. You can also use the keyboard controls:
   - **←** / **→** : Move piece left/right
   - **↓** : Soft drop (move down faster)
   - **↑** : Rotate piece
   - **Space** : Hard drop (instant drop)
   - **P** : Pause/Resume game

## Game Rules

- Stack the falling tetrominoes to create complete horizontal lines
- Completed lines are cleared and you earn points
- Multiple lines cleared at once earn bonus points
- Game ends when pieces stack up to the top

## Scoring

- Lines cleared: 100 points per line
- Multiple lines at once: Bonus multiplier
- Hard drop: 2 points per row

## Features

- Classic 7 tetromino shapes
- Score and line counter
- Start/Pause/Resume button
- Pause functionality (keyboard and button)
- Restart button
- Responsive controls
- Game over detection

Enjoy playing!
